
# Sistema de Gestion y Monitoreo de Redes Wi-Fi

## Descripcion del proyecto

Este proyecto consiste en desarrollar una aplicacion enfocada en el monitoreo y administracion de redes Wi-Fi con fines de seguridad informatica. La aplicacion permitira detectar dispositivos conectados, monitorear actividad sospechosa y generar alertas relacionadas con accesos no autorizados dentro de una red.

La finalidad principal es ayudar a usuarios, estudiantes y pequeñas empresas a tener un mayor control sobre su red mediante una interfaz sencilla y facil de utilizar.

---

# Exposicion del problema

Actualmente muchas personas utilizan redes Wi-Fi sin conocer realmente que dispositivos estan conectados o si existe actividad sospechosa dentro de la red. Esto puede provocar accesos no autorizados, robo de informacion o disminucion del rendimiento de la conexion.

Muchas herramientas de monitoreo existentes son demasiado complejas para usuarios principiantes o requieren conocimientos avanzados de redes y ciberseguridad. Debido a esto, se necesita una aplicacion mas simple e intuitiva.

---

# Plataforma

La aplicacion sera desarrollada principalmente para Windows 11.

## Tecnologias utilizadas

- Python
- SQLite
- Tkinter o PyQt
- GitHub

---

# Interfaz de usuario e interfaz de administrador

## Interfaz de usuario

La interfaz del usuario tendra opciones simples y organizadas como:

- Inicio de sesion
- Panel principal
- Lista de dispositivos conectados
- Historial de actividad
- Alertas de seguridad

## Interfaz de administrador

La interfaz de administrador permitira:

- Gestionar usuarios
- Configurar alertas
- Bloquear dispositivos sospechosos
- Ver reportes del sistema

---

# Funcionalidad

La aplicacion incluira las siguientes funciones:

1. Escaneo de dispositivos conectados a la red.
2. Registro de actividad sospechosa.
3. Generacion de alertas de seguridad.
4. Visualizacion de estadisticas basicas.
5. Inicio de sesion para usuarios y administradores.
6. Almacenamiento de informacion en base de datos.

# Objetivos del proyecto


Desarrollar una aplicacion de monitoreo de redes Wi-Fi que permita mejorar la seguridad y el control de dispositivos conectados.

## Objetivos especificos

- Diseñar una interfaz sencilla e intuitiva.
- Implementar funciones basicas de monitoreo.
- Generar alertas de actividad sospechosa.
- Registrar informacion importante en una base de datos.
